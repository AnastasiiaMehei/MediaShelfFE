## 🚀 Technologies

**Frontend stack:**
- React 19 + TypeScript
- Vite (bundler & dev server)
- Redux Toolkit + React Redux (state management)
- React Router DOM (routing)
- Axios (API requests)
- Tailwind CSS + Tailwind Animate
- Material UI (MUI v9) + Emotion (styled components)
- Framer Motion (animations)
- Swiper (sliders)
- WaveSurfer.js (audio visualization)
- Lucide React & React Icons (icons)

## ⚙️ Installation & Setup

```bash
# Clone repository
git clone https://github.com/your-username/mediashelffe.git

# Navigate to project
cd mediashelffe

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

🔐 Features
User authentication (login/register/logout)

Protected routes for media content

Search and save books, movies, audio, video, and images

Responsive UI with animations

Integration with external APIs (books, movies, pixabay images, video)
